# Catégories de documents administratifs
# Format: Catégorie # Indice pour l'IA
# Les lignes vides et commençant par # seul sont ignorées

# === IMPÔTS ET FISCALITÉ ===
Avis d'imposition                # Document fiscal annuel récapitulant l'impôt sur le revenu dû
Déclaration de revenus           # Formulaire de déclaration annuelle des revenus (2042, etc.)
Taxe foncière                    # Avis d'imposition sur les propriétés bâties/non bâties
Taxe d'habitation                # Avis d'imposition sur le logement occupé
Avis de situation déclarative    # Confirmation de déclaration fiscale (ASDIR)
Attestation fiscale              # Document prouvant la situation fiscale
Crédit d'impôt                   # Notification de crédit ou réduction d'impôt
Prélèvement à la source          # Document relatif au PAS (taux, régularisation)

# === BANQUE ET FINANCE ===
Relevé de compte                 # Relevé bancaire mensuel ou périodique
Relevé d'identité bancaire       # RIB, IBAN, coordonnées bancaires
Relevé de carte                  # Récapitulatif des opérations carte bancaire
Avis d'opération                 # Notification de virement, prélèvement
Contrat de prêt                  # Convention de crédit immobilier ou consommation
Tableau d'amortissement          # Échéancier de remboursement de prêt
Offre de prêt                    # Proposition commerciale de financement
Relevé de livret                 # Relevé de compte épargne (Livret A, LDD, etc.)
Attestation de solde             # Document certifiant le solde d'un compte

# === ASSURANCE ===
Contrat d'assurance              # Police d'assurance (habitation, auto, vie, etc.)
Attestation d'assurance          # Certificat prouvant la couverture
Carte verte                      # Attestation d'assurance auto
Avis d'échéance                  # Appel de cotisation annuelle
Conditions particulières         # Détails spécifiques du contrat
Relevé d'information             # Historique de sinistres (auto)
Déclaration de sinistre          # Formulaire de déclaration d'un dommage
Indemnisation                    # Notification de remboursement sinistre
Quittance d'assurance            # Reçu de paiement de prime

# === EMPLOI ET SALARIAT ===
Bulletin de salaire              # Fiche de paie mensuelle
Contrat de travail               # CDI, CDD, convention d'embauche
Attestation employeur            # Certificat de l'employeur (Pôle Emploi, etc.)
Certificat de travail            # Document de fin de contrat
Solde de tout compte             # Reçu pour solde de tout compte
Avenant au contrat               # Modification du contrat de travail
Attestation de salaire           # Justificatif de revenus employeur
Relevé de carrière               # Historique des emplois et cotisations

# === RETRAITE ===
Relevé de situation individuelle # RSI retraite, points acquis
Notification de retraite         # Décision d'attribution de pension
Attestation de pension           # Justificatif de revenus retraite
Bulletin de pension              # Relevé mensuel de pension

# === SÉCURITÉ SOCIALE ET SANTÉ ===
Carte Vitale                     # Attestation de droits Sécurité sociale
Relevé de remboursement          # Décompte CPAM/mutuelle
Attestation de droits            # Justificatif de couverture maladie
Ordonnance                       # Prescription médicale
Facture soins                    # Facture médecin, pharmacie, hôpital
Arrêt de travail                 # Certificat médical d'arrêt maladie
Attestation CMU                  # Couverture maladie universelle
Notification MDPH                # Décision handicap, AAH, carte invalidité

# === CAF ET PRESTATIONS SOCIALES ===
Attestation CAF                  # Justificatif de droits et prestations
Notification APL                 # Attribution aide au logement
Notification RSA                 # Attribution revenu de solidarité active
Notification prime d'activité    # Attribution prime d'activité
Relevé de prestations            # Récapitulatif des versements CAF

# === LOGEMENT ===
Bail                             # Contrat de location
Quittance de loyer               # Reçu de paiement de loyer
État des lieux                   # Document d'entrée ou sortie
Avis de régularisation           # Régularisation des charges
Attestation d'hébergement        # Certificat d'hébergement à titre gratuit
Acte de vente                    # Acte notarié de propriété
Compromis de vente               # Avant-contrat immobilier
Diagnostic immobilier            # DPE, amiante, plomb, électricité, etc.

# === ÉNERGIE ET SERVICES ===
Facture électricité              # Facture EDF, Engie, fournisseur
Facture gaz                      # Facture de gaz naturel
Facture eau                      # Facture de consommation d'eau
Facture téléphone                # Facture mobile ou fixe
Facture internet                 # Facture box, fibre
Contrat énergie                  # Contrat de fourniture électricité/gaz
Relevé de consommation           # Relevé compteur, estimation

# === VÉHICULE ===
Carte grise                      # Certificat d'immatriculation
Permis de conduire               # Document de conduite
Contrôle technique               # Procès-verbal de visite technique
Facture entretien                # Facture garage, réparation
Contrat de vente véhicule        # Certificat de cession
Certificat de non-gage           # Situation administrative du véhicule

# === IDENTITÉ ET ÉTAT CIVIL ===
Carte d'identité                 # CNI
Passeport                        # Document de voyage
Acte de naissance                # Copie intégrale ou extrait
Livret de famille                # Document d'état civil familial
Acte de mariage                  # Certificat de mariage
Jugement de divorce              # Décision judiciaire de divorce
Certificat de nationalité        # CNF
Titre de séjour                  # Carte de résident, visa

# === ÉDUCATION ===
Diplôme                          # Certificat de réussite (bac, licence, etc.)
Relevé de notes                  # Bulletin scolaire ou universitaire
Attestation de scolarité         # Certificat d'inscription
Attestation de réussite          # Confirmation d'obtention diplôme
Facture scolarité                # Frais d'inscription, cantine

# === JUSTICE ET ADMINISTRATION ===
Jugement                         # Décision de justice
Convocation                      # Convocation tribunal, administration
Décision administrative          # Arrêté, décision préfectorale
Casier judiciaire                # Bulletin n°3
Procès-verbal                    # PV contravention, infraction
Attestation sur l'honneur        # Déclaration personnelle

# === DIVERS ===
Facture                          # Facture commerciale générique
Devis                            # Proposition commerciale chiffrée
Bon de commande                  # Confirmation de commande
Garantie                         # Certificat de garantie produit
Notice                           # Manuel, mode d'emploi
Correspondance administrative    # Courrier officiel, recommandé
